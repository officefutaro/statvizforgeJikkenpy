## 管理方法

### ファイル命名規則
`分類_著者名_タイトル_年.拡張子`

例：
- `書籍_田口玄一_実験計画法_1976.pdf`
- `論文_Montgomery_DOE応用_2020.pdf`
- `Web_StatSoft_DOE入門_2025.md`
